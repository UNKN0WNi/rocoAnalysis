# 洛克王国世界精灵 PVP 阵容分析推荐

## 项目角色

你是一个专注于《洛克王国：世界》PVP 精灵阵容分析推荐的专家。基于属性克制、技能效果和种族值数据，为用户提供阵容搭配建议和克制关系分析。

## 项目文件索引

| 文件 | 用途 |
|------|------|
| `counter.md` | 属性克制关系表 — 包含18种属性的克制/被克制/抵抗关系 |
| `effect.md` | 技能效果指南 — 控制类、伤害类、增益减益、印记类效果速查及主流战术 |
| `rules.md` | 配对推荐分析准则 — 核心推荐规则和队伍构建要点 |
| `output/all_pokemon.json` | 全精灵数据 — 包含种族值、属性、技能、特性等信息 |
| `output/leader_spirits.csv` | 首领精灵列表 — 每队最多1只或以下 |

## 核心推荐准则 (rules.md)

1. **属性克制优先** — 选克制面广、被克制面少的精灵，参考 `counter.md`
2. **高种族值** — 种族值越高越好，结合被动技能综合评估
3. **技能协同** — 高输出 + 增益技能 + 防御/状态应对技能 缺一不可
4. **队伍限制** — 每队6只精灵，首领精灵最多1只
5. **同族限制** — 同一 evolution_chain 视为同一精灵，队伍中不重复出现

## 分析要点

- 以属性克制为框架基础
- 技能协同效果往往比单一精灵强度更重要
- 队伍需包含完整的 **攻防控制** 功能模块

## 属性克制关系摘要 (counter.md)

- **普通系**：克制无，被全属性抵抗
- **草系**：克制火/冰/虫/翼/龙，被水/岩石/地面/毒克制
- **火系**：克制草/冰/虫/机械，被水/地面克制
- **水系**：克制草/电，被火/岩石/地面/机械克制
- **光系**：克制幽/恶，无明显弱点
- **地系**：克制火/电/毒/机械，被草/武克制
- **冰系**：克制火/地面/武/机械，被草/龙/地面/翼克制
- **龙系**：克制草/火/水/电/翼，被龙/冰/妖精克制
- **电系**：克制水/翼，被地面克制
- **毒系**：克制地/恶/幻，被草/萌克制
- **虫系**：克制翼/火，被草/恶/幻克制
- **武系**：克制毒/虫/翼/萌/幽/幻，被普通/地/冰/恶/机械克制
- **翼系**：克制草/虫/武，被电/冰/岩石克制
- **萌系**：克制毒/恶/机械/火/龙，被龙/武/恶克制；抵抗水/火/草
- **幽系**：克制普通/毒/虫/武，被光/幽/幻克制
- **恶系**：克制光/虫/武/萌，被毒/萌/幽克制
- **机械系**：克制地/冰/萌，被火/水/电/武克制；抵抗普通/草/毒/龙/幽/虫/翼/幻
- **幻系**：克制虫/幽，被光/机械/幻克制

## 效果系统要点 (effect.md)

### 下场消失 vs 不消失
- **消失**：控制类(部分)、伤害类(中毒/灼烧)、增益/减益类
- **不消失**：冻结、萌化、寄生、诅咒、星陨、奉献、吸血、全部印记

### 主流战术组合
- **雪影萌化队**：化蝶萌化 → 雪影冻结 → 冰墙减速 → 卡洛儿收割
- **火队**：灼烧叠加 → 击鼓传花 → 陨星虫收割
- **幻系星陨队**：星链叠层 → 多维击打 → 击鼓传花
- **水系湿润队**：湿润 + 光合 = 双能源体系

### 关键应对
- 冻结/冰冻 → 火系/神火系解除
- 萌化 → 逆向演化、红绒十字特性解除
- 印记类 → 倾泻、食腐、焚烧烙印清除

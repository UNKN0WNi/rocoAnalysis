# 《洛克王国：世界》效果指南

## 一、效果分类

| 类别 | 下场消失 | 代表效果 |
|------|---------|---------|
| 控制类 | 部分消失 | 冻结、萌化（不消失）、睡眠、恐惧、混乱、束缚 |
| 伤害类 | 中毒、灼烧（消失） | 寄生、诅咒、星陨（不消失） |
| 增益/减益 | 消失 | 攻击↑、防御↑、速度↑、魔攻↑、魔抗↑、种族值升降 |
| 特殊机制 | 多不消失 | 奉献、吸血、击鼓传花、折返、蓄力、连击 |
| 印记类 | 均不消失 | 中毒/减速/龙噬/风起/湿润/光合/星陨等12种 |

## 二、核心效果速查

### 控制类
- **冻结**：冰系技能，每层扣最大HP×比例，生命值不足时力竭。火系/神火系解除，冰系免疫
- **萌化**：使精灵退化为上一阶形态，种族值降低。雪影娃娃「逆向演化」或「红绒十字」特性解除
- **束缚**：先手值减半，4回合，穿透负面免疫
- **睡眠/冰冻/恐惧/混乱**：限制行动，1-4回合或概率解除

### 伤害类
- **中毒**：每层3%最大HP/回合，毒系/机械系免疫
- **灼烧**：每层2-4%最大HP/回合，层数每回合衰减一半
- **寄生**：每层6-8%最大HP/回合，草系伤害转化为治疗，下场不消失
- **诅咒**：固定25%最大HP/回合，下场不消失
- **星陨**：3回合后结算，每层30威力魔法伤害，下场不消失

### 增益/减益类
属性提升/降低：攻击、防御、速度、魔攻、魔抗。叠加层数×基础值，受PP限制

### 特殊机制类
- **奉献**：强化「啃咬」「虫群」，下场不消失，影响背包精灵
- **吸血**：攻击时恢复每层10%最大HP
- **击鼓传花**：离场时将全部增益/减益传递给下一只精灵
- **折返**：离场换宠，部分技能（冰冻）可穿透折返

### 印记类
| 印记 | 效果 | 清除技能 |
|------|------|---------|
| 减速 | 速度永久-10/层 | 焚烧烙印、倾泻 |
| 龙噬 | 5能耗技能攻击力+40% | 倾泻、食腐 |
| 风起 | 先手威力+20% | 倾泻、食腐 |
| 湿润 | 技能能耗-1/层 | 倾泻、食腐 |
| 光合 | 每回合恢复1能量 | 倾泻、食腐 |
| 星陨 | 伤害触发，每层30威力 | 食腐、疾风天气 |
| 棘刺 | 敌方入场-6%HP | 拔除棘刺 |

**清除技能**：「倾泻」「食腐」清除印记；「净化」清除一般状态

## 三、主流战术

### 雪影萌化队
化蝶萌化降种族值 → 雪影娃娃冻结控场 → 冰墙减速 → 卡洛儿收割

### 火队（击鼓传花）
灼烧叠加 → 击鼓传花传递增益 → 陨星虫收割

### 幻系星陨队
星链快速叠加星陨层 → 多维击打触发结算 → 击鼓传花扩大优势

### 水系湿润队
打湿（湿润印记）→ 光合作用（光合印记）→ 低能耗高频释放

## 四、应对速查

| 效果 | 应对方法 |
|------|---------|
| 冻结/冰冻 | 火系/神火系技能解除 |
| 萌化 | 逆向演化、红绒十字特性 |
| 束缚 | 等待自然消退，用先手技能突破 |
| 中毒 | 净化、毒系/机械系免疫 |
| 灼烧 | 净化、水系/神火系免疫 |
| 印记类 | 倾泻、食腐、焚烧烙印 |

## 五、规则总结

- **叠加**：多数可叠加，攻击印记不可叠加
- **覆盖**：同方只能存在一种同类印记；「吟游之弦」特性可突破此限制
- **下场消失**：中毒、灼烧、属性增益/减益、聒噪等
- **下场不消失**：冻结、萌化、寄生、诅咒、星陨、奉献、吸血、全部印记

## 六、效果组合技巧

- 湿润+光合 = 双能源体系
- 星陨8层以上避免被幻系攻击
- 萌化多层后优先解除
- 击鼓传花时注意传递的减益伤害
